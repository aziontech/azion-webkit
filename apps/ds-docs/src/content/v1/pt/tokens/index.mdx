## Tokens primitivos

Use primitivos quando precisar de valor fixo ou ao definir novas semГўnticas.

| Categoria | Exemplo | Uso |
|----------|---------|-----|
| Neutros | `neutral-50` вЂ¦ `neutral-950` | SuperfГ­cies, bordas, texto sem semГўntico |
| Primary (laranja) | `orange-50` вЂ¦ `orange-950` | Marca; `orange-500` Г© o laranja principal |
| Accent (violeta) | `violet-50` вЂ¦ `violet-950` | Destaques |
| Surface | `surface-0`, `surface-50`, вЂ¦ `surface-950` | Camadas |
| Status | `red-*`, `green-*`, `yellow-*`, `blue-*` | Estados |

**No Tailwind (docs):** O config estende `theme.extend.colors` com `primitives`, `surface`, `primaryBrand` e `accent` de `@aziontech/theme/tokens`. Use `bg-neutral-100`, `text-orange-500`, `border-surface-200`, etc.

**Em JS:**

```js
import { primitives, surfacePrimitives, brandPrimitives } from '@aziontech/theme/tokens';

primitives.neutral[900];
surfacePrimitives.surface[50];
brandPrimitives.primary[500];
```
