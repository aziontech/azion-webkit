## Visão Geral

O componente Fieldset fornece uma maneira de agrupar controles de formulário relacionados. Ele renderiza um elemento HTML nativo `<fieldset>` com recursos de acessibilidade adequados, incluindo um `<legend>` associado para o label do grupo.

Fieldsets são essenciais para criar formulários acessíveis, pois ajudam usuários de leitores de tela a entender a relação entre os controles do formulário.

<Callout type="info" title="HTML Semântico">
  Este componente usa elementos nativos `<fieldset>` e `<legend>`, que fornecem benefícios de acessibilidade integrados sem requerer atributos ARIA adicionais.
</Callout>

## Quando Usar

Use Fieldset quando você precisar:

- Agrupar inputs de formulário relacionados (ex: campos de endereço, informações de contato)
- Criar seções lógicas dentro de um formulário maior
- Melhorar a acessibilidade do formulário com estrutura semântica adequada
- Aplicar um label ou descrição compartilhada para múltiplos inputs
- Desabilitar múltiplos inputs de uma vez

## Acessibilidade

### Navegação por Teclado

| Tecla | Ação |
|-------|------|
| Tab | Move o foco para o primeiro elemento focável dentro do fieldset |

### ARIA

- Use `aria-describedby` para referenciar uma descrição para o grupo do fieldset
- Use `aria-required="true"` quando todos os campos no grupo são obrigatórios

### Notas

- Elementos nativos `<fieldset>` e `<legend>` fornecem acessibilidade integrada
- Leitores de tela anunciam a legenda ao entrar no fieldset
