## Visão Geral

Botões são elementos interativos que disparam ações quando clicados ou tocados. Eles comunicam ações que os usuários podem realizar e são tipicamente colocados em toda a interface, desde formulários até diálogos e navegação.

O componente Button fornece uma maneira consistente e acessível de disparar ações em toda sua aplicação.

## Quando Usar

Use botões para:

- Disparar ações principais (Enviar, Salvar, Continuar)
- Navegar para diferentes visualizações ou páginas
- Alternar estados ou configurações
- Disparar ações secundárias (Cancelar, Excluir)
- Submeter formulários

<Callout type="tip" title="Escolhendo a variante certa">
  Use **primary** para a ação principal, **secondary** para ações alternativas, **destructive** para ações perigosas, e **ghost** para ações menos proeminentes.
</Callout>

## Acessibilidade

### Navegação por Teclado

| Tecla | Ação |
|-------|------|
| Tab | Move o foco para o botão |
| Enter | Ativa o botão quando focado |
| Space | Ativa o botão quando focado |

### ARIA

- Use `aria-disabled="true"` em vez do atributo `disabled` para leitores de tela
- Use `aria-label` para botões apenas com ícone
- Use `aria-busy="true"` quando o botão estiver carregando
