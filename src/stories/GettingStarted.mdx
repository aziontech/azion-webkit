# Easy to use

Esta documentação lhe auxiliará com o projeto [azion-webkit](https://github.com/aziontech/azion-webkit).
O projeto azion-webkit por baixo dos panos utiliza o PrimeVue V3 como biblioteca de componentes básica.

> [Acesse a documentação do PrimeVue](https://v3.primevue.org/).


**projects:**

- [azion-webkit](https://github.com/aziontech/azion-webkit)
- [azion-theme](https://github.com/aziontech/azion-theme)


## Local Development

Instale as dependencias dos projetos Azion Web Kit e Site com `npm install`,
e dentro de Site crie um link para o Azion Web Kit com a instrução `npm link azion-webkit`.

Pronto! Voce está apto para utilizar o `npm run dev` ou `npm run preview` para subir o seu ambiente de desenvolvimento local.


### Realizando modificações no Webkit

Ao realizar modificações no projeto Webkit essas não vão ser refletidas automaticamente no projeto,
a cada modificação é necessário utilizar a instrução `npm run build`.
